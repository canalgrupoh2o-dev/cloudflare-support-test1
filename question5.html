Question 5: Content Not Cached

Response

Dear Customer,

Cloudflare caching is not working on your site due to a `Cache-Control: no-store` header from your server at roskolniv.us, which disables caching.

Steps to Resolve

  • Modify your server configuration to set Cache-Control: public, max-age=3600.
  • In the Cloudflare Dashboard, enable a "Cache Everything" rule for your resources and purge the cache via Purge Cache.

Verify changes and report if the issue continues.

Best regards,
Omir
Your Cloudflare Support Team

Tools I would use:

  • curl - curl -I roskolniv.us to verify headers.
  • Cloudflare Dashboard - dash.cloudflare.com for rules/purge.

Thought Process:

Identified `Cache-Control` header as cause. Recommended server and dashboard adjustments.
